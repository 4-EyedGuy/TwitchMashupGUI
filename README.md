# TwitchMashupGUI (aka Twitch Multi-Stream Mashup)

Удобный (вроде) инструмент для одновременного просмотра нескольких Twitch-стримеров с автопереключением. Сделано для мэшапов совместных стримов с разными point-of-view.

### 🌐 Web-версия (работает прямо в браузере)
👉 **[Открыть веб-версию](https://4-eyedguy.github.io/TwitchMashupGUI/)**

### ✨ Возможности веб-версии
- Запуск двух стримов одновременно
- Автоматическое переключение между стримами

### 💾 Полная версия (Electron)
- Практически бесшовное переключение
- Добавление любого количества стримеров

Для записи, микширования звука и сохранения в MP3 — скачайте Electron-версию из репозитория.

### Как пользоваться
1. Введите ники стримеров
2. Нажмите **«Запустить Мэшап»**
3. Наслаждайтесь!
