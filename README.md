# TwitchMashupGUI (Twitch Multi-Stream Mashup)

Удобный инструмент для одновременного просмотра нескольких Twitch-стримеров с автопереключением.  
Идеально подходит для создания **мэшапов** совместных стримов (POV-мэшапы).

---

### 🌐 Web-версия (прямо в браузере)

👉 **[Открыть веб-версию](https://4-eyedguy.github.io/TwitchMashupGUI/)**

**Возможности веб-версии:**
- Автопереключение + ручное переключение по вкладкам
- Оба стрима загружаются заранее (быстрое переключение)
- Только **2 стримера** одновременно

---

### 💾 Полная версия (Electron)

**Преимущества полной версии:**
- Неограниченное количество стримеров
- Практически бесшовное переключение
- Микширование аудио

---

### Как установить Electron-версию (рекомендуется)

1. Перейдите в раздел **[Releases](https://github.com/4-EyedGuy/TwitchMashupGUI/releases)**.
2. Скачайте последний релиз:
   - `TwitchMashup Setup X.X.X.exe` — **установщик** (рекомендуется)
   - Портативная версия в разработке
3. Запустите программу.

---

### Как пользоваться

1. Введите ники стримеров (в веб-версии — ровно 2)
2. Нажмите **«Запустить Мэшап»**
3. Наслаждайтесь!

Пользуйтесь `автопереключением` или переключайте вручную по вкладкам.

Кнопка `+` добавляет новую строку для ещё одного стримера, и так до бесконечности.

---

### Для разработчиков / запуск из исходников

```bash
git clone https://github.com/4-EyedGuy/TwitchMashupGUI.git
cd TwitchMashupGUI
npm install
npm start
```

### Сборка установщика:
```bash
npm run build
```

Web-версия — это упрощённая демонстрация.
Для полноценной работы (особенно записи) используйте Electron-версию.
